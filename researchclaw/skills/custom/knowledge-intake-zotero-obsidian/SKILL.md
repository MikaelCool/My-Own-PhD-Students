1. Ingest structured sources first: Zotero library, then Obsidian notes, then raw local seed files.
2. Preserve collection names, note titles, and attachment paths as provenance fields.
3. Build a compact digest instead of replaying full documents downstream.
4. Reuse vocabulary from these sources when screening relevance and positioning novelty.

Anti-patterns:
- Ignoring curated local knowledge and jumping straight to web search.
- Dropping provenance, which makes later verification and citation work harder.
